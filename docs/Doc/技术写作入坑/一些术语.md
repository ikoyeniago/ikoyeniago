先简单记录一些推荐了解的术语。

| **术语**                                                     | **缩写** | **含义**                                                     |
| ------------------------------------------------------------ | -------- | ------------------------------------------------------------ |
| Technical Writing                                            | TW       | 技术写作，一些相关术语，请参见[TechScribe Glossary](https://www.techscribe.co.uk/techw/glossary.htm#documentation)、[ClickHelp Software Documentation Glossary](https://clickhelp.com/software-documentation-glossary/)。 |
| Technical Communication                                      | TC       | 技术传播                                                     |
| User Assistance                                              | UA       | 用户帮助                                                     |
| User Experience                                              | UX       | 用户体验                                                     |
| User Interface                                               | UI       | 用户界面                                                     |
| Style Guide                                                  | SG       | 风格指南，用于统一组织内所产出内容的风格，打造品牌形象。     |
| Simplified Technical English                                 | STE      | 简化技术英语，一套指导英语技术写作的方法论。                 |
| [Information Mapping](https://www.tcbok.org/principles-and-practices/information-mapping/) | IM       | 一套区分信息类型以及指导组织信息的方法论。                   |
| Darwin Information Typing Architecture                       | DITA     | 达尔文信息类型化体系结构，OASIS定义的开放标准，用于实现基于XML的文档开发体系结构。 |
| Content Management System                                    | CMS      | 内容管理系统，用于管理开发出的技术文档等内容。               |
| Globalization                                                | G11N     | 全球化                                                       |
| Internationalization                                         | I18N     | 国际化                                                       |
| Localization                                                 | L10N     | 本地化                                                       |
